# Frontend Boilerplate

## Stack
* Package manager: npm
* transpiler: babel
* builder: webpack
* linter: eslint
* formatter: prettier
* html: pug
* css: sass, postcss
* js: ES6+ (ES2015+)

## Usage
Once you start web server by `npm start`, there's an explanation on usage and detail.  
Just see it!
